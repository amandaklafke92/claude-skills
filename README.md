# claude-skills
A collection of skills (structured instruction files) for use with Claude: structured instruction files that teach Claude how to handle specific tasks consistently across conversations.

## Files
Each skill has two files:

- '.md' — read this to see what the skill does before installing
- '.zip' — download this to install the skill in Claude

## How to install

1. Download the '.zip' file for the skill you want
2. In Claude, go to **Customize → Skills**
3. Click **"+"** and select **Upload a skill**
4. Upload the ZIP file and toggle it on
